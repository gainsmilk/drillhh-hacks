## Commit Rules

- NEVER add co-author tags (`Co-Authored-By:`, `Co-authored-by:`, etc) to any commit
- NEVER commit without explicit ask from alxx
- Conventional commits only: `feat|fix|refactor|docs|infra|test|chore(scope): description`
- Subject line imperative mood, under 72 chars
